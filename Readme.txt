# Requirements:

• Update Xcode
• Install cocoapods
	- sudo gem install cocoapods or sudo gem install cocoapods --pre for a pre-release version
		
	The project should be on that directory or you could export the cocoapods path to the project’s directory  
	- export PATH=$GEM_HOME/bin:$PATH
	- pod install 
	- pod repo update 
		- Could take some time on the first build, it will create a new workspace
		- Xcode project that should be run - MovieEmotions.xcworkspace
